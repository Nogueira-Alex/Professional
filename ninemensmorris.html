Nine Men's Morris

Made in 2020

  • Get the Code + Diagrams

Context

This was my final programming assignment for the BTSGP in 2020. I wanted to include at least one pure programming project that's 100% made by me in this Portfolio.

The Game

The Game is local multiplayer and is played in the console. The rules are Nine Men's Morris rules as they are on Wikipedia.

Diagrams

There's UML Diagrams in the Repository that have been made before the game was made. One class diagram and two Activity Diagrams which respectively detail the Move and Mill process.

Design Patterns

As the focus of the course was on Design Patterns I included a bunch of Patterns such as the State, Strategy, Flyweight and Proxy Design Patterns.
